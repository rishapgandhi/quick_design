# Linear Design System
> Category: Productivity & SaaS
> Ultra-minimal, precise, dark-mode-first. Purple accent on near-black canvas.

## Color Palette
- **Background:** `#08090a` (marketing), `#0f1011` (panels), `#191a1b` (elevated)
- **Primary Text:** `#f7f8f8`
- **Secondary Text:** `#d0d6e0`
- **Muted Text:** `#8a8f98`
- **Accent:** `#5e6ad2` (indigo) / `#7170ff` (interactive)
- **Accent Hover:** `#828fff`
- **Border:** `rgba(255,255,255,0.08)` (standard), `rgba(255,255,255,0.05)` (subtle)
- **Success:** `#27a644`

## Typography
- Font: Inter, -apple-system, system-ui, sans-serif
- Weights: 400 (reading), 510 (emphasis), 590 (strong)
- Display: 48-72px, weight 510, letter-spacing -1px to -1.5px
- Body: 15-16px, weight 400, line-height 1.5-1.6
- Headings line-height: 1.0-1.2

## Components
- **Buttons:** `rgba(255,255,255,0.02)` bg, 6px radius, semi-transparent border. Primary: `#5e6ad2` fill.
- **Cards:** `rgba(255,255,255,0.02)` bg, `1px solid rgba(255,255,255,0.08)`, 8-12px radius.
- **Borders:** Always semi-transparent white, never solid dark colors.

## Layout
- Dark backgrounds, content emerges from darkness
- Generous vertical spacing (80px+ between sections)
- Single accent color — everything else is grayscale
- Elevation via background opacity stepping (0.02 → 0.04 → 0.05)

## Agent Rules
- Near-black canvas, NOT white backgrounds
- Brand indigo only for CTAs and interactive elements
- No bold (700) — max is 590
- Aggressive negative letter-spacing on large text
