# Dark Modern
> Category: Starter
> High-contrast dark theme for dramatic presentations and pitch decks.

## Color Palette
- **Background:** `#0d0d0d`
- **Surface:** `#1a1a1a`
- **Elevated:** `#2a2a2a`
- **Primary Text:** `#ffffff`
- **Secondary Text:** `#a0a0a0`
- **Accent:** `#ff6b35` (warm orange)
- **Accent Alt:** `#00d4aa` (teal)
- **Border:** `#333333`
- **Gradient:** `linear-gradient(135deg, #ff6b35, #ff8f65)`

## Typography
- Font: -apple-system, system-ui, 'Segoe UI', sans-serif
- Display: 56-72px, weight 800, line-height 1.0, letter-spacing -2px
- Headings: 32-48px, weight 700, line-height 1.1
- Body: 18-20px, weight 400, line-height 1.6, color `#a0a0a0`
- Use UPPERCASE for labels and small accent text

## Components
- **Buttons:** Gradient fill or solid accent, 8px radius, bold text.
- **Cards:** `#1a1a1a` bg, 1px `#333` border, 16px radius, 32px padding.
- **Badges:** Accent bg, dark text, small rounded pill.
- **Dividers:** 1px `#333` or gradient line.

## Layout
- Full-bleed dark backgrounds
- Dramatic hero with oversized typography
- High contrast between text and background
- Centered content, max-width 1000px for readability
- 100px+ vertical spacing between sections

## Agent Rules
- Bold and dramatic — this is for presentations that need to impress
- Large type, few words per slide
- Use accent color for ONE key element per section
- Dark backgrounds always — never white
- Gradients welcome on hero elements and CTAs
