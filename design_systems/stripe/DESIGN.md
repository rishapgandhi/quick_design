# Stripe Design System
> Category: Fintech
> Clean, professional, trust-inspiring. Purple-blue gradients on white.

## Color Palette
- **Background:** `#ffffff`
- **Surface:** `#f6f9fc`
- **Foreground:** `#0a2540` (dark navy)
- **Secondary Text:** `#425466`
- **Muted:** `#8898aa`
- **Accent:** `#635bff` (Stripe purple)
- **Accent Light:** `#7a73ff`
- **Success:** `#30b130`
- **Border:** `#e6ebf1`
- **Gradient:** `linear-gradient(135deg, #635bff 0%, #80e9ff 100%)`

## Typography
- Font: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif
- Headings: weight 700, dark navy `#0a2540`
- Display: 48-64px, line-height 1.1
- Body: 17-19px, weight 400, line-height 1.6, color `#425466`
- Captions: 14px, color `#8898aa`

## Components
- **Buttons:** `#635bff` fill, white text, 8px radius, 12px 20px padding. Hover: darken 10%.
- **Cards:** White, 1px `#e6ebf1` border, 12px radius, 24px padding, subtle shadow `0 2px 5px rgba(0,0,0,0.08)`.
- **Inputs:** White bg, 1px `#e6ebf1` border, 8px radius, focus: `#635bff` border.
- **Code blocks:** `#f6f9fc` bg, 14px monospace.

## Layout
- Max-width 1080px, centered
- Hero: gradient mesh background or subtle purple tint
- Sections: 96px vertical spacing
- 2-3 column feature grids
- Left-aligned content, generous whitespace

## Agent Rules
- Professional and trustworthy, never playful
- Purple accent used sparingly — mostly on CTAs and highlights
- Dark navy for headlines, medium gray for body
- Subtle gradients OK for hero, not for body content
- Cards should feel elevated but not heavy
